## **vds-repair:** Visual Decision Support for Interactive Repairs

### Local installation:

1. frontend app:
 - new terminal, go to: `src/frontend/`
   - run: `npm install` 
   - run: `npm run dev` to start server or `npm run build` to create static files
 - web vis at `http://localhost:3000`
2. server:
 - run: `npm install` to build the server
 - run: `node server.js` to start server
